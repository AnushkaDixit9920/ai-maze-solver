# 🧠 AI Maze Solver (A* vs BFS)

An interactive Python project that visualizes pathfinding using **A\*** and **Breadth-First Search (BFS)** algorithms in a grid-based maze.

This project demonstrates how intelligent search algorithms find the shortest path efficiently using heuristics and compares informed vs uninformed search.

---

## 🚀 Features

- 🔍 Implementation of **A\*** and **BFS** algorithms  
- 🎮 Real-time visualization using Pygame  
- 📊 Efficiency comparison (nodes explored)  
- 📏 Shortest path length calculation  
- 🎲 Random **solvable maze generation**  
- 🧩 Interactive UI with buttons

## 🧠 Algorithms Used

### 🔵 A* (A-Star Algorithm)
- Uses **heuristic (Manhattan distance)**  
- Formula: `f(n) = g(n) + h(n)`  
- Efficiently finds shortest path by guiding search  

### 🟡 BFS (Breadth-First Search)
- Explores nodes level-by-level  
- Guarantees shortest path but less efficient  

---

## 🎮 Controls

- 🖱️ Mouse Click → Place Start, End, Walls  
- ▶️ Run A* → Find optimal path  
- ▶️ Run BFS → Compare performance  
- 🔄 Reset → Clear grid  
- 🎲 Generate Maze → Create random solvable maze
